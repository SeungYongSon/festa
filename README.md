# Mini Festa

## ⚠ Warning

본 프로젝트가 접근하는 저작권은 실제 서비스를 담당하는 [**Festa**](https://festa.io) 에 있으며,  
이를 유료/상업적인 목적으로 이용 시 저작권 및 법적 책임이 발생할 수 있다는 점을 명시합니다.

## 📄 README

Festa 에 들어가 이벤트들을 확인하고는 싶은데,  
페이지까지 들어가기 귀찬은 사람들을 위해 만들어진 앱입니다.
